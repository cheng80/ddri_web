# DDRI 웹서비스 프로젝트 기획서

작성일: 2026-03-16  
최종 갱신: 2026-03-19  
목적: DDRI 웹서비스의 현재 범위, 대상 사용자, 개발 원칙, 데이터 연동 방향을 최신 상태로 정리한다.

## 1. 프로젝트 개요

DDRI 웹서비스는 강남구 따릉이 대여소 데이터를 바탕으로 아래 2개 목적을 지원한다.

- 일반 사용자:
  - 내 위치 또는 지정 주소 근처에서 특정 시각 기준 대여 가능성을 조회
- 운영 관리자:
  - 자전거 부족 가능성이 높은 대여소를 빠르게 파악하고 재배치 판단 지원

이 서비스는 따릉이 전용 대여 앱이 아니다.  
비로그인 공개 웹으로 제공되며, 조회 중심 정보 서비스로 제한한다.

## 2. 강력 제약

- 이용권 관련 기능 금지:
  - 구매, 조회, 결제, 구독, 이용권 관리
- 사용자 my 기능 금지:
  - 회원가입, 로그인, 프로필, 즐겨찾기, 마이페이지
- 재배치 실행 기능 제외:
  - 관리자 페이지는 판단 지원용이며 실제 재배치 작업은 외부 프로세스에서 수행

## 3. 개발 방식

현재 프로젝트는 `투트랙`으로 진행한다.

- 트랙 A. 분석/모델 산출물 확정
  - 예측 결과 보강
  - 예외 스테이션 규칙 고정
  - 서비스용 출력 필드 정리
- 트랙 B. 웹서비스 개발
  - 화면 구조 설계
  - API 계약 정의
  - Flutter Web 프론트엔드 구현
  - FastAPI 목업 엔드포인트 구성

두 트랙의 접점은 아래 항목으로 제한한다.

- API 응답 스키마
- 예측 결과 연결 포맷
- 예외 스테이션 처리 규칙

## 4. 대상 사용자

### A. 일반 사용자

목표:

- 지금 또는 특정 시각에 내 주변 대여소에서 자전거를 빌릴 수 있는지 확인

핵심 행동:

- 현 위치 찾기
- 주소 검색
- 날짜/시간 선택
- 주변 대여소 조회
- 길찾기 실행

### B. 운영 관리자

목표:

- 부족 가능성이 높은 대여소를 선별하고 재배치 우선순위를 확인

핵심 행동:

- 기준 날짜/시간 설정
- 긴급 대여소 필터링
- 행정동 기준 조회
- 위험도/우선순위 정렬
- 예외 스테이션 분리 확인

## 5. 핵심 페이지 범위

현재 확정된 페이지는 아래 2개다.

### 1. 사용자 페이지

- 경로: `/user`
- 기능:
  - 현 위치 찾기
  - 주소 검색
  - 시간 선택
  - 반경 선택
  - 거리순 대여소 목록 조회
  - 현재 재고, 예측 대여량, 예상 잔여 수 표시
  - 대여 가능 여부 배지 표시
  - 길찾기 버튼

### 2. 관리자 페이지

- 경로: `/admin`
- 기능:
  - 전체 대여소 위험도 목록 조회
  - 기준 날짜/시간 선택
  - 긴급 필터
  - 행정동 필터
  - 정렬 기준 선택
  - 현재 재고 / 예측 수요 / 재고 차이값 표시
  - 위험 점수 및 재배치 우선순위 표시
  - 예외 스테이션 구분 표시

통계 페이지는 현재 범위 밖이다.

## 6. 라우팅 원칙

- 기본 사용자 흐름은 `/user`
- 관리자 화면은 `/admin`
- 루트 `/` 는 사용자 화면과 동일 진입점으로 처리
- 공통 상단 네비게이션은 `사용자 / 관리자`
- `GetX` 기반 라우팅 사용

## 7. MVP 범위

현재 MVP 범위는 아래까지다.

- 사용자 페이지 조회 UI
- 관리자 페이지 목록/필터/정렬 UI
- 목업 API 연동
- 예외 스테이션 메시지 처리
- 반응형 레이아웃
- 공통 테마와 레이아웃 컴포넌트

이번 단계에서 제외하는 항목:

- 로그인/회원가입
- 개인화 저장
- 통계 페이지
- 관리자 메모 저장
- 실제 재배치 실행 기능
- 실제 ML 예측 연동 전까지의 고도화 계산

### 8.2 실전 웹 기준 스테이션

- 강남구 현재 운영 스테이션 전체:
  - 외부 API에서 수집

### 8.3 기타 데이터

- 실시간 재고:
  - `OA-15493 bikeList`
- 예측 데이터:
  - `ddri_work` 산출물 연동 예정
- 예외 스테이션:
  - 실시간 비노출, 비활성, API 장애 등 운영 상태 규칙 반영

## 9. 저장 전략

### 서버 측 저장

- 스테이션 마스터
- 실시간 재고 캐시
- 예측 결과 테이블
- 위험도 스냅샷 또는 조합 계산 결과

### 웹 앱 저장

- 서비스 핵심 기능 기준 저장 기능은 없음
- 로그인, 즐겨찾기, 마이페이지, 사용자 메모는 제공하지 않음
- 단, UX 편의를 위한 로컬 상태 보존은 허용 가능:
  - 최근 주소
  - 마지막 좌표
  - API 베이스 URL 등 개발 편의 설정
