# DDRI 반응형 브레이크포인트 및 레이아웃

작성일: 2026-03-17  
목적: 모바일·태블릿·데스크탑 전 구간 반응형 화면 설계 스펙. Flutter `flutter_screenutil` 및 Stitch 디자인 적용 기준.

> **강력 제약**: DDRI는 따릉이 전용 대여 앱 아님. 비로그인 조회 전용. 이용권·사용자 my 절대 금지.

---

## 0. 반응형 일관성 원칙

> **데스크탑·태블릿·모바일은 반응형이지만 일관성이 있어야 한다.**

### 0.1 시각적 일관성 (공통 유지)

| 항목 | 값 | 비고 |
|------|-----|------|
| Primary 색상 | `#00a857` | 모든 구간 동일 |
| 배경 | `#f5f8f7` | 모든 구간 동일 |
| 폰트 | Public Sans | 모든 구간 동일 |
| 배지 색상 | 대여가능(녹색), 보통(주황), 부족(회색) | 카드·표 공통 |
| 카드 스타일 | rounded-xl, shadow-sm, 흰색 배경 | 모든 구간 동일 |
| 반경 옵션 | 300m, 500m, 1km | 모든 구간 동일 |

### 0.2 정보 구조 일관성

- **검색/입력 영역**: 내 위치 찾기 + 주소 찾기 + 시간대 선택 — 3가지 필수, 구간별 동일
- **대여소 리스트**: 배지(대여가능/보통/부족) + 거리 + 대여소명 + 자전거 수 + 상세보기 + 길찾기(가능하면) — 구간별 동일
- **버튼형 리스트 제약**: 컴팩트 버튼/카드만 쓰면 정보 표시에 한계. **리스트/테이블 형식** 또는 **확장 가능한 카드**로 거리·예상 잔여·주소 등 정보를 충분히 노출
- **지도**: 단일 지도 1개, 원형 반경, 마커 — 구간별 동일
- **상단 네비**: 사용자 / 관리자 (통계 미정) — 구간별 동일 메뉴

### 0.3 구현 시 점검

- 레이아웃만 변경하고, 컴포넌트·색상·타이포는 **공통 Design Token** 사용
- `ThemeData`, `flutter_screenutil` 기준으로 **한 곳에서** 정의 후 구간별로 스케일만 조정

## 1. 브레이크포인트

| 구간 | 최소 너비 | 용도 |
|------|-----------|------|
| **모바일** | 0 ~ 599px | 스마트폰 세로/가로 |
| **태블릿** | 600 ~ 1023px | 태블릿, 작은 노트북 |
| **데스크탑** | 1024px ~ | PC, 큰 모니터 |

### flutter_screenutil designSize 권장

- **모바일 기준**: `designSize: Size(375, 812)` (iPhone 14 기준)
- **태블릿/데스크탑**: 동일 designSize 사용, `MediaQuery` 또는 `LayoutBuilder`로 분기

## 2. 사용자 페이지 반응형 레이아웃

### 2.1 모바일 (0~599px)

```
┌─────────────────────────────────────┐
│  [DDRI 로고]        [≡ 메뉴]        │  ← 상단 바 (햄버거 메뉴)
├─────────────────────────────────────┤
│  [현 위치] [주소 검색]               │  ← 검색 영역 (세로 스택)
├─────────────────────────────────────┤
│  ┌─────────────────────────────┐   │
│  │                             │   │
│  │   단일 지도 (높이 35~40%)    │   │  ← 지도 상단
│  │   원형 반경 + 마커           │   │
│  │                             │   │
│  └─────────────────────────────┘   │
│  [300m] [500m] [1km]  ← 반경 선택   │
├─────────────────────────────────────┤
│  주변 대여소 목록 (총 N개)           │
│  ┌─────────────────────────────┐   │
│  │ 대여가능 150m  DDRI-024      │   │  ← 리스트/카드 1열 (정보 풍부)
│  │ 12대  예상잔여 7대  상세보기  │   │     버튼형보다 정보 표시 우선
│  └─────────────────────────────┘   │
│  ┌─────────────────────────────┐   │
│  │ 보통  320m  DDRI-015         │   │
│  │ 5대   [상세보기 >]           │   │
│  └─────────────────────────────┘   │
│  ...                                │
└─────────────────────────────────────┘
```

- **지도**: 화면 높이의 35~40% (약 200~250dp)
- **카드**: 1열, 전체 폭, 패딩 16px
- **네비게이션**: 햄버거 메뉴 또는 하단 탭

### 2.2 태블릿 (600~1023px)

**가로(landscape)**: 좌우 분할

```
┌─────────────────────────────────────────────────────────────┐
│  [DDRI 로고]     사용자  관리자  통계                        │
├────────────────────────────┬────────────────────────────────┤
│   단일 지도 (좌측 40%)     │  주변 대여소 목록 (2열 그리드)  │
│   [300m][500m][1km]        │  ┌─────────┐ ┌─────────┐       │
└────────────────────────────┴────────────────────────────────┘
```

**세로(portrait)**: 상하 분할 (권장)

```
┌─────────────────────────────────────┐
│  [DDRI 로고]     사용자  관리자      │
├─────────────────────────────────────┤
│  ┌─────────────────────────────┐   │
│  │   단일 지도 (상단 35~40%)    │   │  ← 지도 상단
│  │   [300m] [500m] [1km]       │   │
│  └─────────────────────────────┘   │
├─────────────────────────────────────┤
│  주변 대여소 목록 (총 N개)           │
│  리스트/테이블 형식 (정보 풍부)     │  ← 하단
└─────────────────────────────────────┘
```

- **가로**: 좌측 40% 지도, 우측 60% 리스트
- **세로**: 상단 35~40% 지도, 하단 리스트 (좌우 분할은 가로폭 부족)

### 2.3 데스크탑 (1024px~)

```
┌──────────────────────────────────────────────────────────────────────────┐
│  [DDRI 로고]           사용자  관리자  통계                               │
├────────────────────────────────┬─────────────────────────────────────────┤
│                                │  주변 대여소 목록 (총 N개)              │
│   단일 지도 (좌측 45%)         │  ┌─────────────────────────────────┐   │
│   원형 반경 + 마커             │  │ 카드 1 (전체 폭)                 │   │  ← 1열 또는 2열
│                                │  └─────────────────────────────────┘   │
│   [300m] [500m] [1km]          │  ┌─────────────────────────────────┐   │
│                                │  │ 카드 2                           │   │
│                                │  └─────────────────────────────────┘   │
│                                │  ...                                   │
└────────────────────────────────┴─────────────────────────────────────────┘
```

- **지도**: 좌측 45%, 리스트 우측 55% (기존 Stitch 데스크탑과 동일)
- **카드**: 1열 (넓은 카드) 또는 2열
- **네비게이션**: 상단 가로 메뉴

## 3. 관리자 페이지 반응형 레이아웃

### 3.1 모바일

- **제어 영역**: 세로 스택 (날짜/시간, 군집, 정렬)
- **요약 카드**: 2열 그리드 (작은 카드)
- **표**: 카드형 목록으로 전환 (표 대신)

### 3.2 태블릿

- **제어 영역**: 가로 배치 (flex-wrap)
- **요약 카드**: 2~4열
- **표**: 가로 스크롤 가능한 테이블 또는 카드 2열

### 3.3 데스크탑

- **제어 영역**: 한 줄 가로 배치
- **요약 카드**: 4열
- **표**: 전체 폭 테이블

---

## 4. 공통 컴포넌트 반응형

| 컴포넌트 | 모바일 | 태블릿 | 데스크탑 |
|----------|--------|--------|----------|
| 상단 바 | 햄버거 + 로고 | 로고 + 메뉴 | 로고 + 메뉴 |
| 대여소 리스트 | 리스트/카드 (정보 풍부) | 리스트/테이블 또는 2열 | 1열 또는 2열 |
| 카드 패딩 | 12~16px | 16~20px | 20~24px |
| 폰트 크기 (제목) | 16~18px | 18~20px | 20~24px |
| 폰트 크기 (본문) | 14px | 14~16px | 16px |

### 4.1 대여소 리스트 형식 (버튼형 지양)

- **버튼형 리스트**: 정보 표시에 제약. 지양.
- **권장**: 리스트 행 또는 확장 가능한 카드로 **거리, 대여소명, 현재 자전거 수, 예상 잔여 수, 주소(축약)** 등 충분히 노출
